Getting started

Installing npm

First, to locally host the server, you need to have npm installed. You can do this by first installing the nvm package manager with the following command

curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

Then exiting and reopening your terminal and typing:

nvm install node

Creating a New Project

We will use the package create-react-app to build our basic project framework.

npx create-react-app my-app
cd my-app
npm start

This will create a file tree that looks something like this:

.
├── README.md
├── node_modules
├── package-lock.json
├── package.json
├── public
└── src

To host the server: 1. npm install to install all depedencies (these are listed in package.json) 2. npm start

For more information, see Creating a New React App.

Adding a new page

To add a new page, follow the below steps: 1. Add a path to src/App.js. To do this, add the line <Route path="/[endpoint]" element={<[your page] />} />. 2. At the top of src/App.js, add the line import [your page] from "./pages/[your page].js" 3. Create a file in src/pages called [your page].js.

When in doubt, look at the other routes in src/App.js as an example.

Creating the page

Suppose we are creating a page called “Page”, your src/pages/Page.js file should a bit like this

const Page () => {
  return (
    //some html
  );
}
export default Page

When in doubt, look at src/pages/Layout.js as an example.
