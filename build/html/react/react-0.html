Getting started

Installing npm

First, to locally host the server, you need to have npm installed. You can do this by first installing the nvm package manager with the following command:

curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

Then exiting and reopening your terminal and typing:

nvm install node

Creating a New Project

We will use the package create-react-app to build our basic project framework.

npx create-react-app my-app
cd my-app
npm start

This will create a file tree that looks something like this:

.
├── README.md
├── node_modules
├── package-lock.json
├── package.json
├── public
└── src

The public folder contains any data you want to include on your site (think images, data files, etc). The src folder contains all the scripts we will use to generate our page.

Within the src folder, there is a file App.js. This our “main” page, or the script that is run with the command npm start.

To host the server:

  1. npm install to install all depedencies (these are listed in package.json)

  2. npm start

For more information, see Creating a New React App.
